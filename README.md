# Smart Student Management System (C)

A menu-driven student record management system written in **C**, demonstrating dynamic data structures, file persistence, and record manipulation.

## Summary
Implemented a console-based Student Management System using **singly linked lists** to store records and **file I/O** for persistence. The project focuses on core C programming concepts important for embedded and systems development.
